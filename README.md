# 12306-Ticket-booking
负责一条线路，十个站点的订票。程序可以模拟随机数量的人口预定随机始发站到终点站的火车票。在自行订票时，可以显示余票数。同时可以浏览火车在离开站台0-10之后的座位分布图(ASCII特殊符号来代表每个人的身份）
